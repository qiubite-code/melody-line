一、系统认识：它能帮助学生看见什么

本系统不是专业录音棚软件，而是面向音乐课堂的“声音—图像—谱例”学习工具。它把学生演唱、教师范唱或上传音频转化为旋律线，让学生直观看到音高、力度、节奏、休止、乐句与谱例之间的关系。

音乐元素系统表现课堂理解重点
音高 / 旋律线条在 Y 轴上下移动：上行上扬，下行下沉。观察旋律走向、音区变化、级进、跳进与音准偏差。
力度 / 音量线条粗细与透明度变化：强音更粗更实，弱音更细更淡。理解强弱、渐强、渐弱和情绪推动。
节奏 / 节拍线条按真实时间横向推进，点位疏密体现声音变化频率。观察快慢、密集与舒展，配合小节线理解节拍。
休止 / 换气声音低于阈值时断线留白，并可形成休止标记。理解停顿、换气、句逗和乐句结束。
乐句划分系统依据较长停顿自动显示虚线分隔和“第几句”。分析问答句、重复句、变化句与音乐结构。
简谱 / 五线谱可将已绘制旋律线转换为课堂参考谱例。把“听到的旋律”和“看到的谱面”对应起来。

二、界面认识：四个主要区域

新版界面按课堂使用逻辑分为功能区、参数设置区、观察区和转谱结果区。操作手册按钮已内置在页面中，点击即可在线阅读。

区域主要功能使用建议
功能区开始录音、暂停录音、结束录音、上传音频、保存图片、生成谱例、导出谱图、操作手册。课前准备和功能切换时使用。
参数设置区休止阈值、线条平滑、音域范围、乐句停顿、调号、拍号、速度等。建议课前调好，课中少量微调。
观察区显示当前音高、唱名/音名、力度、乐句、旋律线、小节线和播放光标。课堂投屏时重点观看。
音频播放区系统只保留一个真实播放器，可播放、暂停、拖动进度条。用于回放录音或上传音频，黄色光标同步移动。
转谱结果区显示简谱和五线谱参考结果。适合课后分析、导出谱图和学习评价。

三、学生快速上手：3 分钟会操作

步骤学生要做什么观察重点
1. 打开系统使用浏览器打开网页，等待界面加载完成。确认能看到功能区、观察区和操作手册按钮。
2. 开始录音或上传音频点击“开始录音”，听完预备拍后演唱；或上传 mp3/WAV 后播放。录音前保持安静，听到提示音后进入第一小节。
3. 观察旋律线边唱边看线条变化，注意高低、粗细、断开、乐句和小节。上扬=音高升高；下沉=音高降低；断开=休止或换气。
4. 回放与拖动录音结束后用播放器回放，可拖动时间条查看任意位置。完整旋律线保留，黄色光标显示当前播放位置。
5. 保存或生成谱例点击“保存图片”或“生成简谱/五线谱”。可用于小组展示、作业反馈和前后对比。

学生操作口诀：听预备拍，稳进入；看高低，辨强弱；遇留白，找休止；看光标，对回放；生成谱，做比较。

四、教师课堂流程：课前、课中、课后

阶段教师操作教学目的
课前准备打开网页，测试麦克风和播放器；设置调号、拍号、BPM、休止阈值和音域范围。保证演示稳定，减少现场调试。
导入环节播放或演唱一句熟悉旋律，让学生描述线条走向。建立“声音—线条—音乐元素”的对应关系。
示范环节教师点击“开始录音”，听预备拍后范唱。让学生观察音高、力度、停顿和乐句。
练习环节学生或小组轮流演唱，系统生成旋律线。即时反馈音准、呼吸、节奏和力度。
回放分析拖动播放器时间条，结合黄色光标定位某一处。分析关键音、跳进、休止和乐句连接。
评价环节保存旋律图，生成简谱/五线谱，对比第一次与调整后演唱。形成过程性评价证据。

五、录音功能：开始、暂停、继续、结束

新版将“开启麦克风”改为“开始录音”，更符合课堂使用流程。开始录音后会先播放一小节预备拍提示音，预备拍结束后才正式拾音和绘制旋律线。

按钮 / 状态功能说明课堂建议
开始录音申请麦克风权限，播放预备拍后进入正式录音。演唱者听到预备拍后从第一小节进入。
暂停录音临时暂停拾音和绘制，已生成线条保留。讲解、纠错、换气练习或临时中断时使用。
继续录音从暂停处继续同一次录音和线条绘制。适合分句练习或教师边讲边录。
结束录音停止麦克风并生成可回放的录音音频。结束后可拖动播放器回看完整旋律线。
观察区同步按钮观察区也有录音按钮，并与功能区按钮状态同步。投屏讲解时不用回到页面上方操作。

录音时请尽量保持环境安静，演唱者距离麦克风约 20—50 cm。声源太远会导致识别不稳，声源太近可能出现喷麦尖峰。

七、观察区读图方法

看到的现象可能代表什么教师追问
线条整体向上旋律上行，音高逐渐升高。这句音乐是否更明亮、更有推进感？
线条整体向下旋律下行，音高逐渐降低。这里是否有结束感、叹息感或回落感？
线条突然大跳可能是真实跳进，也可能是噪声或泛音误判。听觉上是否真的跳了？是否需要调整麦克风或阈值？
线条变粗变亮声音增强，力度变大。这里是否是重音、高潮或情绪加强？
线条变细变淡声音减弱，力度变小。是否出现渐弱、收束或远去感？
线条断开留白休止、换气、停顿或声音过弱。这个停顿是句中呼吸，还是乐句结束？
黄色光标移动当前音频播放位置。现在听到的声音对应图上的哪一段？

八、简谱 / 五线谱生成与谱面设置

点击“生成简谱 / 五线谱”后，系统会根据已绘制的旋律线生成课堂参考谱例。谱例适合单声部旋律参考，复杂伴奏、多声部合唱需要教师人工校对。

设置项可选内容使用建议
调号C、G、D、A、E、B、F♯、F、B♭、E♭、A♭、D♭、G♭ 等。根据教材歌曲或课堂调式选择。
升降号类型自动匹配、升号调、降号调、无升降。一般选择自动；讲解调号时可手动切换。
调式大调、小调、Ionian、Dorian、Phrygian、Lydian、Mixolydian、Aeolian、Locrian。适合拓展民族调式、教会调式和旋律色彩。
拍号2/4、3/4、4/4、5/4、6/8、7/8，也可自定义。与歌曲实际节拍一致，方便小节线显示。
速度Largo、Adagio、Andante、Moderato、Allegro、Vivace、Presto + BPM。用于预备拍和谱例速度标注。
导出谱图将当前谱例保存为图片。可用于课件、作业反馈和课堂评价。

十、参数调节指南：让线条更稳定、更准确

调参建议按顺序进行：先调休止阈值，再调音域范围，再调线条平滑，最后调横向速度、乐句停顿和谱面设置。不要一次改太多项。

参数它控制什么推荐值 / 调法
休止阈值判断声音是有效音还是休止/噪声。普通课堂 0.012—0.025；嘈杂教室 0.025—0.040；声音小可降低。
线条平滑控制线条是否抖动。演唱教学建议 0.88—0.95；想看滑音细节可降低。
最低音 / 最高音控制显示和识别音域。男声可 60—900Hz；女声/童声可 100—1600Hz；乐器可适当扩展。
横向速度控制旋律线在画面上的展开程度。普通课堂 80—110 px/s；节奏观察 120—160 px/s。
乐句停顿判断多长的停顿算新乐句。歌曲 0.60—0.90s；朗读或短句可 0.40—0.70s。
音名吸附 / 稳线模式减少细碎抖动，让线条贴近半音刻度。课堂展示建议打开；精细研究颤音或滑音时可关闭。

音高识别采用浏览器端实时算法，会受麦克风质量、距离、环境噪声、人声泛音和伴奏干扰影响。课堂中应把它作为“可视化反馈工具”，重要结论仍需结合听觉判断。

十一、常见问题与排查

问题可能原因解决方法
点击开始录音没有反应浏览器未授权麦克风，或页面不是 HTTPS / localhost。允许麦克风权限；用 HTTPS 或 localhost 打开；检查系统输入设备。
唱了但没有旋律线休止阈值过高、声音太小、麦克风太远。降低休止阈值；靠近麦克风；提高演唱音量。
没人唱也有线条环境噪声进入识别。提高休止阈值；关闭风扇/空调直吹；调整麦克风方向。
线条出现冲天竖线喷麦、辅音、起音瞬态或高频噪声被误判。离麦克风 20—50 cm；避免直接吹麦；开启稳线模式。
C4 被识别成 C3 或八度错误人声泛音和基频周期混淆。保持钢琴音高校准模式；限制合适音域；使用外接麦克风。
上传音频播放卡顿音频过大、浏览器压力高或重复播放器占用资源。使用新版单播放器；换 Chrome/Edge；尽量使用压缩后的 mp3。
拖动播放器后线条消失旧版会重新绘制。使用新版：拖动时保留完整线条，只移动黄色光标。
生成谱例不准确复杂伴奏、多声部或噪声影响主旋律识别。尽量使用单声部演唱/乐器音频；生成后教师校对。

教师调参口诀：线条乱，先降噪；不断线，调阈值；不居中，调音域；太抖动，加平滑；看节奏，开采样；讲乐句，看虚线；听回放，看光标。
